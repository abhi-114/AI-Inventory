# AI-Driven Retail Demand & Inventory Orchestrator

## 📌 Project Overview
This is a B.Tech Final Year Major Project designed to bridge the gap between Machine Learning and Retail Supply Chain operations. It follows a modular architecture:
- **Module 1:** Leakage-Safe Data Layer
- **Module 2:** Decision-Oriented Feature Layer
- **Module 3:** Hybrid Ensemble Learning (Lasso, RF, LightGBM)

## 🚀 Features
- **Interactive Dashboard:** Real-time Store/Product drill-down.
- **Profit/Loss Estimator:** Live financial health tracking.
- **What-If Simulator:** Interactive price elasticity testing.
- **Model Comparison:** Evaluates multiple algorithms side-by-side.

## 🛠️ Installation
1. Clone the repo: `git clone <your-repo-link>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run main.py`
